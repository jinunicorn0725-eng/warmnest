# 暖巢 Warmnest

> *A Home For Every Soul · 给每一个流浪的灵魂，一个温暖的归处*

一个专注于流浪动物救助与领养的公益网站原型。从街角的偶遇到温暖的归属，把每一次驻足都接住，把每一份善意都看见。

---

## 项目简介

**暖巢 Warmnest** 是一个面向普通大众的流浪动物救助与领养公益平台概念原型，旨在探索通过数字化手段连接"发现者、志愿者与潜在领养人"三方的可能性。

平台围绕两条核心路径展开：

- **救助路径**：路人发现流浪动物 → 拍照上报 → 数据入库 → 志愿者跟进
- **领养路径**：浏览待领养动物 → AI智能匹配 → 申请领养 → 接它回家

> 本项目为课程作业原型，重点验证产品逻辑与交互体验，部分功能以模拟数据呈现。

---

## 核心功能

### 一、首页 `index.html`
- 全屏 Hero 区域，情感化主视觉与标题呈现
- 待领养动物预览卡片（三列网格展示）
- 平台数据展示（救助数量、成功领养、志愿者人数）
- 志愿者招募入口
- 各功能页面导航跳转

### 二、流浪动物上报 `report.html`
- 四步流程引导（发现 → 记录 → 上报 → 跟进）
- 完整上报表单：照片上传（拖拽 / 点击）、动物种类、发现时间、具体位置、状态描述、联系方式
- 数据持久化：图片上传至 Supabase Storage，记录写入 PostgreSQL 数据库
- 提交后即时更新"最近上报"列表

### 三、待领养页面 `adoption.html`
- 待领养动物卡片展示，含编号、名称、性格描述
- 四步领养流程说明（浏览 → 申请 → 面谈 → 接它回家）
- 领养须知与"终身承诺"提示模块
- **AI 智能匹配**：用户填写居住情况、生活节奏、相处偏好、饲养经验后，AI 以流式输出方式生成个性化宠物推荐建议

## 技术栈

| 类别 | 技术 |
|------|------|
| 结构 | 原生 HTML5（无框架，纯静态） |
| 样式 | 原生 CSS（CSS Variables 主题、Grid 布局） |
| 交互 | 原生 JavaScript（零依赖） |
| 字体 | Cormorant Garamond / Noto Serif SC / Inter |
| 动画 | IntersectionObserver、CSS Keyframes |
| 数据库 | Supabase（PostgreSQL + RLS 行级安全） |
| 文件存储 | Supabase Storage（`animal-photos` 桶） |
| AI 模块 | Claude API（流式输出，SSE 直连） |
| 部署 | GitHub Pages |

## 开发工具及模型

| 工具 / 模型 | 用途 |
|-------------|------|
| **Claude Code** | 核心 AI 辅助开发工具，底层接入 Kimi-K2.5 模型，通过自然语言描述需求自动生成页面结构、交互逻辑与样式代码 |

| **VS Code** | 代码编辑与本地开发环境 |
| **Chrome DevTools** | 页面布局调试与响应式适配优化 |

## 设计语言

### 色彩

```
--bg:        #FAF6F1   米白底（主背景）
--bg-deep:   #EAE0D5   暖米（次级背景）
--gold:      #C9A66B   主金色（强调色）
--gold-dark: #A8854B   深金色
--deep:      #5E503F   深咖色
--text:      #22333B   主文字色
```

### 字体层级

- **Cormorant Garamond Italic**：英文标题、装饰性短语，传递古典优雅气质
- **Noto Serif SC**：中文主标题与正文重点，衬线字体增添人文温度
- **Inter**：导航、按钮、表单等功能性文字，清晰易读

### 排版习惯

- 中英对照：中文标题下配一行斜体英文作为呼应
- 节标题前置小字 eyebrow + 长划线作为章节引导
- 大背景衬底字（半透明装饰字）为页面提供呼吸感与层次感
